## Technology Used

- TypeScript
- React.js
- Redux
- Tailwind CSS
- React-hook-form
- Linter
- Prettier
- React-quill


## Setup

- Download or Clone this project 
- Run `npm install` to download dependencies
- In .env.local file , configuration id can be changed.
- npm run dev
- Go to [http://localhost:3000](http://localhost:3000) on browser
